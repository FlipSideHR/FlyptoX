Easy Configuration

It's *almost* as easy as cloning the repo from git, and running ./flyptox setup.

Develop with confidence.

The exchange is built from the ground up with TDD in mind. Gulp, mocha, chai, sinon, karma - all part of the standard build process. Because the system is built with a TDD focus, adding new tests is a breeze.

Scalable

FlyptoX is made to scale. Run it on a single machine, or scaled out to multiple api, database, and marketEngine nodes as needed.

Add any currency pair!

Out of the box FlyptoX comes loaded with the currencies: BTC, USD. And the currency pair: BTC/USD. You can add as many currencies, and pairs as you like.
